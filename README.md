# Global-Airbnb-Performance-Dashboard

## 1. Purpose

This project focuses on analyzing Airbnb listings data using Power BI. The goal of this project is to explore different aspects of Airbnb hosts, listings, and review performance to understand patterns in the data.

## 2. Tech Stack

1. Data Visualization
Microsoft Power BI – Used to design interactive dashboards and visualizations.
2. Data Analysis Language
DAX (Data Analysis Expressions) – Used for creating calculated measures, KPIs, percentages, and cumulative metrics.
3. Data Source
CSV Dataset – Used as the primary dataset for analysis of Airbnb listings and host information.
4. Data Processing
Power Query (ETL) – Used for data cleaning, transformation, and preparation before analysis.
5. Version Control
Git – Used for version control and tracking project changes.
6. Project Hosting
GitHub – Used to store and share the project repository.
7. File Storage
Google Drive – Used to store and share the large .pbix dashboard file.

## 3. Data Source

- Source: Maven Analytics

- Dataset Name: Airbnb Listings Dataset

- Format: CSV


## 4. Features
### Business Problem
Airbnb platforms generate large amounts of listing and host data. However, it becomes difficult to quickly understand patterns such as host verification status, listing distribution, and review performance.
 

### Goal of the Dashboard
The main goal of this dashboard is to analyze Airbnb listings and host information to uncover useful insights.
The dashboard helps in:
- Understanding verified vs non-verified hosts
- Analyzing hosts with and without profile pictures
- Tracking review trends over time
- Exploring listing distribution across locations

### Key Visuals
 
#### 1.  KPI Cards 
- Total listings
  
- verified hosts

#### 2. Line Chart

#### 3. Bar Charts
– Comparison of verified and non-verified hosts

#### 4. Location Analysis 
– Distribution of listings across different areas

#### 5. Percentage Metrics 
– Host profile and verification analysis
 

## 5. Business Impact & Insights
The dashboard provides valuable insights such as:
1. Identifying the percentage of verified vs non-verified hosts
2. Understanding how host profile pictures relate to credibility
3. Observing review trends over time
4. Analyzing which locations have the highest number of listings
These insights can help businesses improvehost management, build trust with customers, and make better strategic decisions.
